# TaskDuty

A local single-user MERN task manager built with React, TypeScript, TailwindCSS, Express.

## Features

- Create, read, update, and delete tasks.
- Track title, description, due date, category, and completion status.
- Validate all required fields on the frontend.
- Reject due dates in the past.
- Filter by category and completion status.

## Project Structure

```text
client/   React + TypeScript + TailwindCSS frontend
```

## Local Setup

1. Install dependencies:

```bash
npm run install:all
```

```bash
CLIENT_ORIGIN=http://localhost:5173
```

```bash
npm run dev
```
